# CLAUDE.md

Global Claude Code instructions are organized in `~/.claude/rules/`.

- `interaction.md`: response style, environment, URLs, Notion scope.
- `safety.md`: secrets and conclusive-operation boundaries.
- `workflow.md`: atomic work units and review gates.
- `coding.md`: implementation, testing, exceptions, comments, EOF, ternaries.
- `subagents.md`: model and subagent routing preferences.
- `config-management.md`: where this global config is maintained.

Rules load automatically. Do not duplicate rule content here.
